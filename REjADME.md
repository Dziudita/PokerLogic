# PokerLogic

PokerLogic is a prototype Texas Hold'em poker engine built for testing poker game logic.

The goal of this project is to develop the core mechanics of poker separately from any full platform or UI.

This allows the poker logic to be tested, improved, and later integrated into a larger web application.

## Project Goals

- Build a working Texas Hold'em game engine
- Support 2 players
- Generate and shuffle a 52-card deck
- Deal cards to players and the table
- Evaluate poker hands
- Determine the winner
- Keep the logic modular so it can be reused in other projects

## Tech Stack

- TypeScript
- Next.js (for testing UI)
- TailwindCSS (optional for styling)
